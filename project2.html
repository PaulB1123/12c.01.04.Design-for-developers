Colors and animations

The purpose of the exercise is to apply a color pallet to a mockup landing page design, then add animations.

This project was dived into three parts where the focus was to create a complete mockup with an entirely new style where the colors are changing.

The first part was to apply a color palette, second one was to create a dark theme and the last one was to add animations

As an overall of the project I have learned the importance of using var when designing a page in CSS and how to use animations

You can check the website here

The first part of the assignment was to implement a color palette

For the color palette, I wanted to use a palette of colors that are easy to see and are not that tiring for the eye, after a couple of tests, I have decided to stick with the orange palette for its simplicity

Implementing the palette with UI elements

Once I had the colors ready, we tested it on some UI elements to see if the contrast between the colors would be high enough.

The second part of the assignment was to create a dark theme

The Light Mode is the default option and the main color is the light orange making the text stand out and the navigation where there is a switch button

The Dark Mode can be displayed by clicking the button from the navigation and it will change the entire colors of the pages into only black/white and gray colors

The third part of the assignment was to add animations

The animation that I added was for the h1, where I split all the letters from the words, and than I made a transition for each letter to appear one after another with a delay.
